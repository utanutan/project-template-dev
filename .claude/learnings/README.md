# Agent Learnings

各エージェントが過去のプロジェクトで得た知見を蓄積するディレクトリ。

## 仕組み

1. ユーザーから指摘・修正指示があった場合、PMが該当エージェントにフィードバックを伝達
2. 各エージェントは指摘内容を分析し、再発防止のための知見を自分のファイルに追記
3. 次回以降のプロジェクトで、エージェントは起動時にこのファイルを参照

## ファイル命名規則

`<agent-key>.md` 形式（例: `senior-coder.md`, `review-guardian.md`）

## 知見の記録フォーマット

```markdown
## YYYY-MM-DD: [カテゴリ] タイトル

**指摘内容**: ユーザーからの指摘の要約
**原因分析**: なぜこの問題が発生したか
**改善策**: 今後どうすべきか
**適用条件**: どのような状況でこの知見を適用すべきか
```
